# AFTERBURN 2.0

AFTERBURN is an exercise lifecycle intelligence platform for military, emergency management, law enforcement, homeland defense, and interagency response organizations.

This repository contains a deployable Next.js application (App Router) with a premium dark UI and demo-mode lifecycle workflows.

## Prerequisites

- Node.js 20+
- npm 10+

## Install

```bash
npm install
```

## Local development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the project in Vercel.
3. Framework preset: **Next.js** (auto-detected).
4. Build command: `npm run build`.
5. Output: default Next.js output.
6. Deploy.

No private API keys are required for this demo mode build.
